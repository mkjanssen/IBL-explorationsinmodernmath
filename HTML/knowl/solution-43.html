Solution 6.1.1.1.
  1. Just using the recursive formula provided in , we could find the number of years it would take Joris to accumulate 500 board games by generating the terms of the sequence and seeing when it passes 500. We would do this by first finding \(G_1\text{,}\) then \(G_1\) to find \(G_2\text{,}\) then using \(G_2\) to find \(G_3\text{,}\) etc.
  2. This is highly inefficient! In order to know, e.g., \(G_{500}\text{,}\) you would need to compute all preceding terms.
  3. An explicit formula is given by \(G_t = 37 + 8t\text{.}\) Observe that \(G_1 = 37 + 8\cdot 1 = 45\text{,}\) \(G_2 = 37 + 8\cdot 2 = 37 + 16 = 53\text{,}\) and \(G_3 = 38 + 8\cdot 3 = 61\text{.}\)
